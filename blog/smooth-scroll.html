CROSSBROWSER SUPPORT CSS ONLY SMOOTH PAGE SCROLL (SIMPLE STEPS)

Author:Mbokazi Siphamandla

Typical page smooth scroll make use JavaScript smooth effect on page scroll, but this can be done easily with just few lines of CSS. In this demonstration I will show you how to build a smooth scroll effect for page scroll.

VIEW DEMO DOWNLOAD SOURCE

I. HTML

1. Empty anchor tags with unique IDs

We will start by creating at least three anchor tags with unique IDs, id="section1",id="section2",id="section3", respectively. Later we will have at least three elements to be scroll, and those three elements will be having the same IDs mentioned above( which are id="section1",id="section2",and id="section3".These anchor tags are not visible in the DOM since there are empty.

NOTE: these anchor tags must be placed in the same parent element as elements that will be scrolled.

2. Top Navigation menu anchor tags

we create a menu with at least three anchor tags which has href attribute that targets the IDs of those anchor tags mentioned above which were id="section1">,id="section2",id="section3">, respectively. When the user click on these links, the page will be scrolled to section whose ID is same as the one in that navigation anchor tag. We then use target selector and transform:translate() and transition:transform; to create smooth animation on page scroll.

3. Top Navigation menu anchor tags

The last anchor tag is fixed at bottom and right corner of the viewport by using position fixed which keeps the anchor link at the same position even when the page is scrolled. position fixed allow use to make use right:5px; bottom:50px,to place the code the link at the bottom of the page. This will have same href as home anchor link which returns the user to the start of the web page.

3. Content boxes to be scrolled (Having same IDs as anchor tags at top)

The next step is to create at least three sections containing content (all may have same class name: class="content"),these sections will be scrolled by the user. These sections have same IDs as those anchor tags we started with, which are id="section1", id="section2",and id="section3" ,respectively. We put these sections inside a parent element with the class name .content-wrap.We put some h1 content for demonstration.

II. CSS

2. Hide the empty anchor tags

We will simply start by hiding the empty anchor tags, to avoid any possible empty space for these empty tags. 

  
         /*hide empty tags*/
            a#section1,
            a#section2,
            a#section3{ display:none;}

1. Full screen content boxes

Each section has some height, in this case we use height:100%; width:100% for all sections, some heading and unique background colour for demonstration. The height is used later with transform property when we animate the scroll effect. Then we put backface-visibilty:hidden transition: transform 0.6s ease-in-out; which will help to create smooth effect on page scroll. The last we put transform: translateZ( 0 ); to initialize the transform property for transition:transform; to work. 

   
         /*Unique colour for each content section*/
        .content-wrap #section1{ background:blue;} 
        .content-wrap #section2{ background:black;}
        .content-wrap #section3{ background:red;} 
                                           
        /*All three content sections share this class*/
        .content{
                width: 100%; /* full screen width*/
                height: 100%; /* You choose height */
               z-index:0; /*behind the top navigation when page is scrolled*/
               transform: translateZ( 0 );/*Initialize transform property for*/
               transition: transform 0.6s ease-in-out;
               backface-visibility: hidden; /*avoids bad transition effects*/
        }

2. Smooth Scroll using target Selector (and transform property)

When the user click any the links in the navigation bar, the ID in with the same name as the one in the href is targeted, which is in those the invisible anchor tags, we target that relevant ID and to animate the jump to a section with that same ID, and use translateY to scroll the page to that particular section. Remember the transform property is animated. 

   
         /* Smooth CSS3 Scrolling */
         a[ id= "section1" ]:target ~ .content-wrap .content{
        -webkit-transform: translateY( 0);
        transform: translateY( 0 );
        }
        a[ id= "section2" ]:target ~ .content-wrap .content{
            -webkit-transform: translateY( -100% );
            transform: translateY( -100% );
        }
        a[ id= "section3" ]:target ~ .content-wrap .content{
            -webkit-transform: translateY( -200% );
            transform: translateY( -200% );
        }
